# Flowbit

**Flowbit** is a visual diagramming tool for building systems using nodes and connections.

It allows users to create structured diagrams quickly by placing nodes on a canvas and linking them with arrows.

Inspired by tools like Eraser, Excalidraw and tldraw, Flowbit focuses on **visual thinking for developers and system designers**.

---
## Tech Stack

- **React**
- **TypeScript**
- **TailwindCSS**
- **ReactFlow**
- **Zustand**

---

## Getting Started

### Install dependencies
`npm install`

### Run development server

`npm run dev`
